## Tecnologias Utilizadas

- TypeScript
- ReactJS
- Sass

##  Instalando e Executando

Clonando o repositório:

```
git clone https://github.com/thainarapenha/portfolio.git
```

Instalando todos os pacotes:

```
npm install
```

Rodando o projeto:

```
npm run dev
```
